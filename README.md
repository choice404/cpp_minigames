# Games

## Tic Tac Toe
You can find the Tic Tac Toe game found in /tictactoe directory
To play the game, first compile the main.cpp file and run the executable

![Tic Tac Toe](./readme_assets/tictactoe.gif)

Linux/Mac
```sh
g++ main.cpp
./a.out
```
